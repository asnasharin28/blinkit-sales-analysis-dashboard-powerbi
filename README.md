## Project Overview
This project analyzes Blinkit grocery sales data to understand sales performance, product trends, and outlet behavior using **Power BI**.  
Data cleaning and transformation were performed using **Power Query** before building the dashboard.

---

## Dashboard Preview
![Blinkit Dashboard](images/Blinkit%20Dashboard.png)

## Dataset
The dataset contains item-level and outlet-level information such as sales, ratings, item attributes, and outlet details.

---

## Data Cleaning & Transformation (Power Query)
Data preparation was done in **Power Query**, including:

- Replacing inconsistent values  
- Text formatting and standardization  
- Data type corrections  

---

## Power BI Dashboard
The dashboard highlights:

- Total sales, average sales, item count, and average rating  
- Sales trends by outlet establishment year  
- Sales by item type and fat content  
- Performance by outlet size, type, and location  
- Interactive filters for analysis
